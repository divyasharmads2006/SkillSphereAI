# 🚀 SkillSphere AI

SkillSphere AI is a full-stack skill-sharing platform where users can connect, exchange skills, send learning requests, schedule meetings, and grow together.

## ✨ Features

- 👤 User Registration & Login
- 🎯 Personalized Dashboard
- 📚 Skill Sharing
- 🔍 Search Skills
- 🤝 Send & Receive Requests
- ✅ Accept/Reject Requests
- 👨‍💼 User Profiles
- 📊 Dashboard Statistics
- 💾 MySQL Database Integration
- 🎨 Modern Responsive UI

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Java
- Database: MySQL
- Server: Java HTTP Server

## 📂 Project Structure

```
backend/
frontend/
lib/
```

## ⚙️ How to Run

1. Clone the repository.
2. Import the project into VS Code.
3. Start MySQL.
4. Run `BackendServer.java`.
5. Open `frontend/login.html` in your browser.
6. Enjoy SkillSphere AI!
